Инструкция для создания телеграм-бота, конвертирующего написанный или вставленный текст в файл озвучки в расширении .ogg.

1. Для начала, нужно получить учетную запись в Телеграме, если её у вас нет. Затем в поиске ввести BotFather и зайдя на страницу, зарегистрироваться и получить свой ключ для бота. 
Ключ нужно будет вставить в файле bot.py в строке 4(TOKEN) вместо текста.
https://telegram.im/BotFather
Про все настройки бота в самом Телеграм, там есть инструкция.

2. Затем скопировать все файлы с githab и установить в программу, в которой вы работаете. Pycharm или Visual Studio Code, например.

3. Затем запустить в терминале команду python -m venv venv, которая создает папку venv.

4. Написать строку pip install -r requirements.txt
более подробная информация про пакет pyttsx3:
https://learn4kid-python.firebaseapp.com/python_data_structure/python_speech_synthesis/#%D1%83%D1%81%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BA%D0%B0-%D0%BF%D0%B0%D0%BA%D0%B5%D1%82%D0%BE%D0%B2-%D0%B2-windows

5.Установить ffmpeg по ссылке:
https://community.chocolatey.org/packages/ffmpeg
или командой choco install ffmpeg

6. Написать в командной строке Терминала python bot.py.

7. Теперь можно пользоваться ботом в Телеграме.

8. Инструкцию настройки голоса можно найти в интернете, настройки файлов телеграм-бота также.

9. Также нужно будут создать папку в корневом каталоге, где будут создаваться временные аудиофайлы.
