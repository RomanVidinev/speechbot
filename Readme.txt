Инструкция для создания телеграмм-бота, конвертирующего написанный или вставленный текст в файл озвучки в расширении .ogg.

1. Для начала, нужно получить учетную запись в Телеграме, если её у вас нет. Затем в поиске ввести BotFather и зайдя на страницу, зарегистрироваться и получить свой ключ для бота. 
Ключ нужно будет вставить в файле bot.py в строке 4(TOKEN) вместо текста.
Про все настройки бота в самом Телеграмм, там есть инструкция.

2. Затем скопировать все файлы с githab и установить в программу, в которой вы работаете. Pycharm или Visual Studio Code, например.

3. Затем запустить в терминале команду python -m venv venv, которая создает папку venv.

4. Написать строку pip install -r requirements.txt

5. Запустить файл bot.py.

6. Теперь можно пользоваться ботом в Телеграме.

7. Инструкцию настройки голоса можно найти в интернете, настройки файлов телеграм-бота также.

8. Также нужно будут создать папку data в корневом каталоге, где будут создаваться временные аудиофайлы.
